Projects & Achievements

Smart Energy Meter

The Internet of Things (IoT) presents an efficient and cost-effective way to transfer energy consumption information wirelessly, allowing users to monitor electricity usage and automatically generate bills. The primary objective of this project is to measure electricity consumption in home appliances using IoT.

Technologies Used:
  • Real-time Monitoring
  • Mobile Application
  • User Convenience

Integrating Flexible and Lightweight Solar Panel into BIO-SMART Watch

This project involves integrating flexible, lightweight solar panels into a BIO-SMART watch, enabling it to harness solar energy for charging. This renewable energy source extends the watch's battery life while promoting eco-friendly technology.

Technologies Used:
  • Lightweight Design
  • Durability
  • User Convenience
